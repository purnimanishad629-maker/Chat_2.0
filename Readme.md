# 🌟 Purnima's Motivation Hub with AI Assistant 🤖

A beautiful, responsive motivational website with a Smart AI Assistant that can answer questions, provide motivation, and help with studies.

## ✨ Features

- **🏠 Interactive Home Page** - Beautiful card-based layout with motivational content
- **❤️ Shayari Collection** - Heart-touching Hindi poetry
- **📜 Motivational Poems** - Inspiring poems in Hindi
- **💬 Inspirational Quotes** - Famous quotes from great personalities
- **🌼 Positive Thoughts** - Daily positive affirmations
- **🤖 Smart AI Assistant** - ChatGPT-like assistant for:
  - General questions
  - Study help (Math, Science, etc.)
  - Motivation and guidance
  - Friendly conversations
